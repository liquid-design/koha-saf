stap 7
# Maak het swapbestand van 2GB aan
sudo fallocate -l 2G /swapfile

# Zet de juiste permissies (alleen root kan lezen/schrijven)
sudo chmod 600 /swapfile

# Formatteer het bestand als swap
sudo mkswap /swapfile

# Activeer de swapfile
sudo swapon /swapfile

# Controleer dat swap actief is
sudo swapon --show
free -h

# Voeg toe aan /etc/fstab zodat swap bij herstart automatisch geladen wordt
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab

step 9
Set up Apache
Koha requires mod_rewrite to be enabled within Apache in order to run.
Typically this can be enabled with:

# rm /etc/apache2/sites-available/default-ssl.conf 
# rm /etc/apache2/sites-available/000-default.conf 
# a2enmod rewrite cgi headers proxy_http
# systemctl restart apache2
